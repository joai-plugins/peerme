# PeerMe

Connect PeerMe to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/peerme) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-peerme@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-peerme --url https://cortex.joai.ai/mcp/apps/peerme
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/peerme.git ~/.cursor/plugins/local/joai-peerme
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-peerme&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWktcGVlcm1lIjp7InVybCI6Imh0dHBzOi8vY29ydGV4LmpvYWkuYWkvbWNwL2FwcHMvcGVlcm1lIn19fQ==) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/peerme
```
